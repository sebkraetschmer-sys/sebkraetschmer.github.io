1. Verantwortlicher

Sebastian Krätschmer · Weinstraße 6a · 35510 Butzbach
E-Mail: apps.kraetschmer@gmail.com

2. Lokale Datenspeicherung

Alle Daten (Ausgaben, Budgets, Kontostände, Personennamen) werden ausschließlich lokal per SwiftData gespeichert. Keine Serverübertragung, kein Entwicklerzugriff.

3. Gerätefunktionen

  • Kamera (OCR): Fotos von Kassenbons werden lokal per Apple Vision Framework analysiert. Keine Übertragung an Dritte.

  • P2P-Synchronisation: Daten werden verschlüsselt direkt via MultipeerConnectivity übertragen. Kein Server zwischengeschaltet.

4. Keine Tracking-Tools

Keine Analyse-Software, keine Werbenetzwerke, keine Nutzerprofile.

5. Ihre Rechte

  • Auskunft & Export: Daten jederzeit in der App einsehen und als CSV/Excel exportieren
  • Löschung: App deinstallieren oder einzelne Einträge entfernen
